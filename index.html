Organize seus arquivos. Rapidamente. Pelo terminal.

Ryzor é uma ferramenta CLI em Python para organizar, agrupar e fazer backup de arquivos por tipo/extensão — simples, rápida e personalizável.

Ver no GitHub Como começar

Começar (Quickstart)

Rápido exemplo para ver o Ryzor em ação — execute no diretório do projeto:


# Organizar arquivos
python src/cli.py organize -p ./meus_arquivos -d ./organizados

# Definir novas extensões
python src/cli.py define -t Imagens -exts .jpg .png .gif

Ideal para

Usuários que querem organizar pastas rapidamente usando regras por extensão.

Requisitos

Python 3.11+

Instalar

pip install rich pyfiglet

O que faz

  • Define categorias por extensões e realoca arquivos automaticamente.
  • Preview das mudanças antes de executar (modo seguro).
  • Opção de backup (send2trash) e barra de progresso com Rich.
  • Persistência simples em JSON (extensions.json).

Dependências

  • Python 3.11+
  • rich
  • pyfiglet
  • send2trash
